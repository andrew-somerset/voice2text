# voice2text — GM Windows prototype

Windows-first local dictation with an optional, permission-aware Ask Glean route.

- Hold the configured trigger key to dictate locally into the focused application.
- Double-tap the trigger to begin an Ask Glean recording.
- Tap the trigger a third time to stop, transcribe locally, and submit only the final text.

This `gm_dev` branch is currently a design-only greenfield project. See [CLAUDE.md](CLAUDE.md) for the Windows architecture, security boundaries, and ordered implementation milestones.
